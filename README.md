# Eliza CLI

A command-line interface for the Eliza framework built with Ratatui in Rust.

## Features

- Rich terminal user interface
- Command system with tab completion
- Plugin system
- Real-time updates
- Concurrent state management

## Getting Started

```bash
cargo run
```

## Development

```bash
cargo test
cargo run --example custom_command
```
